# Euro Park Parking Management System

A full-stack web application for managing parking space applications
in the Euro Park residential community.

## Tech Stack

### Backend

- Python 3.12+
- FastAPI
- SQLAlchemy 2.x
- Pydantic
- Alembic
- Microsoft SQL Server
- T-SQL

### Frontend

- Nuxt 4.4.8
- Vue 3
- TypeScript

### Install dependencies

```powershell
cd frontend
npm install
```

### DevOps

- Docker
- Docker Compose
